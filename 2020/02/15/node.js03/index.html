mongoDB数据库增删改查

mongoDB NoSql
mongoDB
发布日期:   2020-02-15

mongoDB数据库

又称为NoSql数据库,即非关系型数据库,适用于处理大数据,采用json格式来存储数据，契合javascript语法操作，因此很适合node.js的开发者

操作mongoDB可以通过node.js中提供的数据库API来进行操作，也可以使用mongoD的可视化操作软件来进行操做

数据库中一些概念

一个数据库软件中可以包含多个数据仓库，每一个数据仓库中可以创建多个数据集合来存放不同类别的数据，每一集合中存放了同一类的一组数据叫文档，由json对象的形式存取，采用键值对的方式存取，对象的属性就叫字段

使用node.js操作mongodb

下载第三方模块mongoose,使用connect方法连接数据库,首先需要在命令行中启动mongdDB

net stop mongdodb //停止mongodb服务
net start mongodb //开启mongodb服务

connect方法中有两个参数,第一个是数据库的地址类似url地址，第二个是选项,这个方法返回的是promise对象,再mongdodb中不需要使用什么命令创建数据库，如果你连接的数据库不存在，会自动创建该数据库

数据库的增删改查

node中与数据库相关的api的都是异步的,所以支持promise的方法获取回调结果

方法一利用实例对象插入文档

  1. 首先需要创建集合用来存放具体的数据,用model()方法,返回的是一个构造函数,用实例对象的方法来操作数据
  2. 创建集合之前需要规定集合的规则即规定集合中数据属性和其数据类型,用mongodb.Schema构造函数创建
  3. 像集合中插入数据,即创建文档,new一下model()方法返回的构造函数,写入要插入的数据按照集合规则来写,最后保存
const mongoose = require('mongoose');

mongoose.connect('mongodb://localhost/playground', {useNewUrlParser: true, useUnifiedTopology: true})
    .then(() => console.log('数据库连接成功'))
    .catch(err => console.log(err,'数据库连接失败'));

//创建集合的规则,即规定需要加入的数据的数据类型
const cs = new mongoose.Schema({
    name: String,
    author: String,
    publish: Boolean
})

//创建集合
const Course = mongoose.model('Course', cs);//两个参数集合名称和集合规则，返回构造函数,利用实例对象的方法操作集合中的数据

//创建文档,像集合中插入数据
const course = new Course({
    name: 'node.js基础',
    author: '曹强',
    publish: true
});
//使用实例对象的save方法保存数据
course.save();

方法二是利用构造函数的create方法

//通过create()方法,两个参数第一个是插入的数据,第二个回调返回插入数据的结果和错误信息
Course.create({name: 'javascript', author: '曹强', publish: false}, (err, doc) => {
    //错误信息
    console.log(err);
    //当前插入的文档
    console.log(doc);
})

//通过promise对象操作异步结果
Course.create({name: 'javascript', author: '曹强', publish: false})
    .then(doc => {
        console.log(doc);
    }).catch(err => {console.log(err);})

导入数据到数据库中

适用于json文件中的数据

mongoimport -d 数据库名称 -c 集合名称 --file 要导入的数据文件    //使用该命令前首先要将mongoimport.exe的路径添加到系统环境变量中

查询文档

//查找集合find()返回promise对象,查找一组数据,以数组形式返回
// User.find().then(result => console.log(result)).catch(err => console.log(err));

//查找一个对象findOne(),默认返回第一个数据的对象,只会查找一个
User.findOne().then(result => console.log(result)).catch(err => console.log(err));

//两个方法都可以传对象参数作为查找条件
//$gt匹配大于 $lt匹配小于   匹配age大于20并且小于50的
User.find({age: {$gt:20, $lt:50}}).then(result => console.log(result));
//$in匹配是否包含
User.find({hobbies: {$in: ['足球']}}).then(result => console.log(result));
//select()方法查询字段,不同字段之间空格隔开,默认会查询_id字段在前面加-排除
User.find().select('name age -_id').then(result => console.log(result));
//sort()根据什么字段进行排序,前面加-为降序不加则为升序
User.find().sort('age').then(result => console.log(result));
User.find().sort('-age').then(result => console.log(result));
//skip()跳过多少数据 limit()限制查询数量
User.find().skip(2).limit(3).then(result => console.log(result));

删除文档

//删除满足条件的一个数据,如果有多个满足条件的删除第一个,返回删除的那个数据
User.findOneAndDelete({}).then(result => console.log(result));

//根据条件删除多个数据,如果不传条件或传空对象默认删除所有
//返回一个对象{n:删除的个数,ok:删除是否成功(0/1),deletedCount:删除的个数}
User.deleteMany({}).then(result => console.log(result));

更新数据/修改

//根据条件更新一个数据,返回一个对象{ n: 1, nModified: 1, ok: 1 }
// User.updateOne({name: '张三'}, {name: '张飞'}).then(result => console.log(result));

//根据条件更新多个数据,返回操作成功的对象
User.updateMany({}, {age: 30}).then(result => console.log(result)).catch(err => console.log(err));

验证字段是否满足规则，限定字段的一些规则

用catch方法捕捉验证的错误信息

Post.create({title:'aa', age: 60, category: 'java', author: 'bd'})
    .then(result => console.log(result))
    .catch(error => {
        // 获取错误信息对象
        const err = error.errors;
    //errors存着所有有错误的字段中的信息
        // 循环错误信息对象
        for (var k in err) {
            // 将错误信息打印到控制台中
            console.log(err[k]['message']);
        }
    })

集合的关联

在想要关联的字段规则的type属性中指定想要关联字段的IDmongoose.Schema.Types.ObjectId, ref:关联的集合

联合查询用populate()指定要查询的关联字段即可
